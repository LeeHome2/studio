# **App Name**: Nutrition Guide

## Core Features:

- Image Analysis: Analyzes uploaded food images to identify the meal.
- Ingredient List Generation: Lists main ingredients found within a given meal. The LLM tool should reason about possible ways the meal was prepared, and decide if common preparations call for an ingredient or not.
- Nutrition Facts Generation: Using the image analysis and ingredient list, estimates the nutritional values (calories, protein, fat) of the identified food item and presents in Korean.
- Diet Recommendation: Give diet recommendation in Korean depending on how balance or imbalanced food appears to be.
- Upload UI: Implement drag and drop component with loading progress to upload photo of food item

## Style Guidelines:

- Primary color: A muted, natural green (#8FBC8F) to evoke feelings of health and nature.
- Background color: A light, desaturated off-white (#F5F5DC) to provide a clean and calming backdrop.
- Accent color: A warm, analogous yellow (#BDB76B) to highlight key information and call to actions.
- Font: 'PT Sans', a humanist sans-serif, for both headings and body text to combine a modern look with a touch of warmth.
- Use simple, clean icons to represent nutritional information, ingredients, and diet tips.
- A clean and intuitive layout with clear sections for image display, nutritional information, and diet recommendations.
- Subtle transitions and animations to provide feedback and enhance the user experience.
